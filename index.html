02 NODE CONFIGURATION
🪝
Webhook Input
n8n · Webhook Trigger
Receives POST with video script (text), screenshots (base64), and Pinterest board URL.
method: POST
path: "/storyboard"
auth: Header Token
body: multipart/form-data
📄
Script Parser
n8n · Code Node (JS)
Splits script into scenes by timestamp or paragraph. Extracts dialogue, action lines, and shot descriptions.
split_by: paragraph / [SCENE]
output: [{scene, text, type}]
max_scenes: 20
👁️
Claude Vision
HTTP · Anthropic API
Analyses uploaded screenshots — identifies visual style, character expressions, environment, and lighting mood.
model: claude-opus-4-6
input: base64 images
extract: style, mood, faces
📌
Pinterest Scraper
HTTP · Pinterest API v5
Fetches pins from your Pinterest board. Extracts dominant colors, typography styles, and animation aesthetic references.
endpoint: /boards/{id}/pins
extract: colors, fonts, mood
limit: 24 pins
🎭
Emotion Analyser
HTTP · Claude AI
Reads the full script + visual context. Maps each scene to an emotion arc: joy, tension, melancholy, triumph, suspense, calm.
output: [{scene, emotion, intensity}]
arc_type: hero / narrative
format: JSON
🎵
YT Music Matcher
HTTP · YouTube Data API v3
Uses emotion tags to search YouTube for free-to-use background music. Returns 5 options per scene with BPM and mood metadata.
query: {emotion} + "no copyright"
filter: creativeCommons
per_scene: 5 tracks
🧠
Storyboard Generator
HTTP · Claude AI (Main)
The core node. Takes all inputs — script scenes, vision output, emotion map, Pinterest style — and generates complete storyboard JSON with Figma annotations.
model: claude-sonnet-4-6
prompt: storyboard_system.txt
output: structured JSON
🎨
Figma Formatter
n8n · Code Node (JS)
Transforms Claude's output into Figma Plugin JSON schema — with frames, text layers, component IDs, motion arrow vectors, and color styles.
schema: figma-plugin-v2
frame_size: 1920×1080
components: auto-named
📤
Webhook Delivery
n8n · HTTP Request (POST)
Sends the final compiled JSON bundle to your custom webhook URL. Includes storyboard, music list, emotion map and Figma import file.
method: POST
body: full_bundle.json
header: X-Workflow-ID

05 HOW TO USE IN FIGMA
1
Run the n8n Workflow
POST your script + screenshots to your webhook URL. The workflow runs all nodes and delivers the JSON bundle to your custom endpoint.
2
Copy the Figma JSON
Grab the figma_json field from the webhook response (or copy it above). This contains all frames with layers, annotations and notes.
3
Install "JSON to Figma" Plugin
In Figma, go to Plugins → search "JSON to Figma" or "Figma Tokens". Paste the JSON and click Import — frames auto-generate.
4
Check Emotion + Music List
The music_list in the bundle gives you 5 free YouTube tracks per scene, matched to the detected emotion arc of your script.
5
Review Pinterest Style Refs
Each frame includes a Pinterest tag pulled from your board. Use these as visual style anchors when creating the final animation assets.
6
Hand Off to Animator
Each frame has motion type, duration, easing, face expression guide, camera shot, and typography spec — everything your animator needs.
